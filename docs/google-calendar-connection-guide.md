# 구글 캘린더 연동 가이드

## 개요

인터뷰 스케줄링 자동화 기능은 **채용담당자(관리자/리크루터)**가 실행하며, 일정 생성/초대 발송은 **채용담당자(주최자) 구글 계정 권한**으로 수행됩니다.

- 채용담당자(주최자)는 **구글 캘린더 연동이 필수**입니다.
- 면접관은 **우리 서비스 가입/구글 연동이 없어도** 이메일만으로 **캘린더 초대(이벤트 참석자 초대)**를 받을 수 있습니다.
- 다만 “공통 가능 시간 자동 계산(바쁨/FreeBusy 조회)”을 위해서는, **면접관 캘린더가 채용담당자 구글 계정에 공유되어 있어야** 합니다.

## 연동 방법

### 1. 가입 시 자동 연동 (권장)

새로 가입하는 사용자는 가입 완료 후 자동으로 구글 캘린더 연동 페이지로 이동합니다.

1. 회원가입 완료
2. 구글 캘린더 연동 페이지 자동 이동
3. "구글 캘린더 연동하기" 버튼 클릭
4. 구글 계정 선택 및 권한 승인
5. 연동 완료

### 2. 기존 사용자 연동

이미 가입한 사용자는 다음 방법으로 연동할 수 있습니다:

1. 대시보드 접속
2. `/dashboard/connect-calendar` 페이지로 이동
3. "구글 캘린더 연동하기" 버튼 클릭
4. 구글 계정 선택 및 권한 승인
5. 연동 완료

### 3. 직접 URL 접속

브라우저에서 다음 URL로 접속:
```
/api/auth/connect-google-calendar
```

## 필요한 권한

구글 캘린더 연동 시 다음 권한이 필요합니다:

- **구글 캘린더 읽기 권한**: 면접관들의 일정을 확인하여 공통 가능 시간을 찾기 위해 필요합니다.
- **구글 캘린더 쓰기 권한**: 면접 일정을 생성하고 수정하기 위해 필요합니다.
- **이메일 주소 확인**: 사용자 식별을 위해 필요합니다.

## 연동 확인

연동이 완료되면 다음을 확인할 수 있습니다:

1. 대시보드에서 인터뷰 스케줄링 자동화 기능 사용 가능
2. 면접관으로 선택 시 에러 없이 진행 가능
3. 구글 캘린더에 면접 일정이 자동으로 생성됨

## 문제 해결

### "모든 면접관이 구글 캘린더에 연동되어 있어야 합니다" 에러

이 문구는 정책 변경으로 더 이상 사용하지 않습니다.

대신 아래 상황에서 자동화가 중단될 수 있습니다.

#### 1) “면접관/회의실 캘린더의 바쁨 시간을 조회할 수 없습니다”

- **원인**: 해당 면접관(또는 회의실) 캘린더가 **채용담당자(주최자) 구글 계정에 공유되어 있지 않음**
- **해결**:
  - 면접관이 자신의 구글 캘린더를 **채용담당자 구글 계정에 공유**(바쁨 시간 조회 가능하도록)하거나
  - Google Workspace 정책/권한(조직 정책상 공유 제한 등)을 확인해주세요.

### 연동이 안 되는 경우

1. 브라우저 캐시 삭제
2. 시크릿 모드에서 재시도
3. 구글 계정 로그아웃 후 재로그인
4. 관리자에게 문의

## 참고

- 연동 정보는 안전하게 암호화되어 저장됩니다.
- 언제든지 설정에서 연동을 해제할 수 있습니다.
- 연동 해제 시 인터뷰 스케줄링 자동화 기능을 사용할 수 없습니다.
